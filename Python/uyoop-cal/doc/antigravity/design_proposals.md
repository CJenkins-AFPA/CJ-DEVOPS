# Propositions Design UX/UI : uYoop

Voici deux concepts générés pour visualiser la nouvelle direction artistique "Dark/Neon DevOps".

## Concept 1 : Login Page Immersive
L'objectif est de créer une rupture totale avec le calendrier vide actuel. Une entrée "solennelle" et immersive dans le système.

![Concept Login](/home/cj/.gemini/antigravity/brain/84a070c0-5e0a-4591-aa45-c0abd1fda6f7/login_page_concept_1767900603658.png)

**Points clés :**
- **Background** : Ambiance "Matrix" subtile, très sombre.
- **Focus** : Uniquement le formulaire.
- **Esthétique** : Verre fumé (Glassmorphism) et Néon vert.

---

## Concept 2 : Dashboard "Mission Control"
Une fois connecté, l'utilisateur atterrit sur un tableau de bord opérationnel, pas juste un calendrier.

![Concept Dashboard](/home/cj/.gemini/antigravity/brain/84a070c0-5e0a-4591-aa45-c0abd1fda6f7/dashboard_page_concept_1767900622660.png)

**Points clés :**
- **Structure** : Cartes aérées avec titres clairs.
- **Info** : Pipeline de déploiement visuel (KPIs) en premier plan.
- **Couleurs** : Fond Gris Anthracite profond (#0D0D0D) pour faire ressortir le vert sans agresser les yeux.
